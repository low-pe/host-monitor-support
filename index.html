HostMonitor Support

HostMonitor is an iOS app for monitoring your vast.ai GPU hosting fleet. Track your machines, earnings, and rental status in real-time from your iPhone.

Getting Started

  1. Download HostMonitor from the App Store
  2. Obtain a read-only API key from your vast.ai account settings
  3. Enter your API key in the app when prompted
  4. Start monitoring your GPU fleet!

Features

  • Real-time balance and earnings tracking
  • Complete machine fleet overview
  • Detailed machine specifications (GPU, CPU, network, storage)
  • GPU-based rental tracking
  • Sort and filter machines
  • Discord webhook notifications
  • Auto-refresh with customizable intervals

Requirements

  • iOS 17.0 or later
  • Active vast.ai host account
  • Read-only API key from vast.ai

Privacy & Security

Your API key is stored securely in iOS Keychain and never shared. All communication uses HTTPS encryption. HostMonitor uses a read-only API key and cannot make changes to your vast.ai account.

Frequently Asked Questions

Is my API key safe?

Yes. Your API key is stored in iOS Keychain (Apple's secure credential storage) and is never transmitted anywhere except to vast.ai's official API over HTTPS.

Why does the app request a read-only API key?

A read-only API key ensures the app can only view your data and cannot make any changes to your vast.ai account, machines, or settings.

How often does the app refresh data?

You can set auto-refresh intervals from 1 minute to 1 hour in the Settings. You can also manually refresh at any time by pulling down on the dashboard.

Can I monitor multiple vast.ai accounts?

Currently, HostMonitor supports one API key at a time. You can change your API key in Settings if you need to switch accounts.
